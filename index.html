Exoplanet Visualizer Data Repo

CP8318 - Data Source

Please download the data sets below in order to run the project files.
You will download a zip file titled, "project.zip".
Once you have downloaded the data, please follow the steps below, which are
also specified in the readme file in the submission.

  1. Extract the contents of project.zip
  2. Put the separately submitted code files and places them all under project/code/
  3. Run the file main.py

The project structure should look like so:

../project/
code/
main.py
naive_bayes.py
preprocess_koi.py
utils.py
data/
cumulative_koi_data.csv
habitable_planets_detailed_list.csv
non_habitable_planets_confirmed_detailed_list.csv
